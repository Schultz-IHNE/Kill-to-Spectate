# Kill to Spectate

A simple server-side Minecraft mod that changes player death behavior.

## Features

- If a player is killed by another player, they are switched to Spectator mode.
- Deaths from mobs, fall damage, lava, or other environmental causes do not trigger spectator mode.
- Lightweight and server-side only.

## Use Cases

Perfect for:
- PVP servers
- Hardcore-style events
- Last man standing game modes
- SMP tournaments
